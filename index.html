How it works?

1. Ask with context Ask in natural language. Start by selecting a location on the map, using your current location, or choosing a campus preset, so the agent can understand where your journey begins.
2. Choose the right workflow The agent decides whether your question needs tools, including navigation, route comparison, weather, transport status, and web search. Tools are called only when specific live information is needed.
3. Get actionable results The model uses the route, weather, and other data it receives to give practical advice. For navigation requests, it shows an interactive route preview with station and line details when available.
